# Standard Library

The LOGOS standard library provides core types and utilities for all programs.

## Definition

A Unit is a record.

A Pair of [A] and [B] has:
    a first, which is A.
    a second, which is B.

A Triple of [A] and [B] and [C] has:
    a first, which is A.
    a second, which is B.
    a third, which is C.
