# Core

Core types and algebraic structures for LOGOS programs.

## Definition

A Result of [T] and [E] is either:
    An Ok (value: T).
    An Err (error: E).

An Option of [T] is either:
    A Some (value: T).
    A None.

A List of [T] is a generic collection.

A Map of [K] and [V] is a generic collection.
