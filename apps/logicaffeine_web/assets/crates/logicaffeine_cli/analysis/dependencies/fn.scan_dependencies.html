Function scan_dependencies

pub fn scan_dependencies(source: &str) -> Vec<Dependency>
Expand description

Scans the first paragraph (Abstract) of a LOGOS file for [Alias](URI) links.

The Abstract is defined as the first non-empty block of text following the module header (# Name). Links inside this paragraph are treated as imports. Scanning stops at the first empty line after the abstract or when a code block header (##) is encountered.

# My Game

This module uses [Geometry](file:./geo.md) for math.

Let x be 1.
