1//! Logicaffeine Web Application
2//!
3//! Browser-based IDE for LOGOS language learning, built with Dioxus.
4//!
5//! # Architecture
6//!
7//! The app is structured around a gamified learning experience that teaches
8//! first-order logic through progressive exercises and spaced repetition.
9//!
10//! ## Data Flow
11//!
12//! ```text
13//! ┌─────────────────────────────────────────────────────────────────────┐
14//! │                           User Interface                            │
15//! │  Landing → Learn → Studio → Profile                                 │
16//! └─────────────────────────────────────────────────────────────────────┘
17//!                                   │
18//!                                   ▼
19//! ┌─────────────────────────────────────────────────────────────────────┐
20//! │                         Content Pipeline                            │
21//! │  content (JSON) → generator (templates) → Challenge                 │
22//! └─────────────────────────────────────────────────────────────────────┘
23//!                                   │
24//!                                   ▼
25//! ┌─────────────────────────────────────────────────────────────────────┐
26//! │                        Learning Loop                                │
27//! │  Challenge → grader (validation) → game (XP, combos) → feedback     │
28//! └─────────────────────────────────────────────────────────────────────┘
29//!                                   │
30//!                                   ▼
31//! ┌─────────────────────────────────────────────────────────────────────┐
32//! │                       Persistence Layer                             │
33//! │  progress → storage (LocalStorage) → srs (scheduling)               │
34//! │                   ↓                                                  │
35//! │            achievements (unlocks) → unlock (module access)          │
36//! └─────────────────────────────────────────────────────────────────────┘
37//! ```
38//!
39//! ## Module Responsibilities
40//!
41//! | Module | Purpose |
42//! |--------|---------|
43//! | [`content`] | Loads curriculum from embedded JSON, provides era/module/exercise hierarchy |
44//! | [`generator`] | Fills exercise templates with lexicon words, creates [`Challenge`](generator::Challenge) instances |
45//! | [`game`] | Manages XP, streaks, combos, level progression, and exercise flow |
46//! | [`grader`] | Validates answers with normalization (whitespace, Unicode equivalence) |
47//! | [`progress`] | Tracks completed exercises, scores, and review state |
48//! | [`srs`] | Implements SM-2 spaced repetition algorithm for review scheduling |
49//! | [`achievements`] | Defines and checks achievement conditions, awards badges |
50//! | [`unlock`] | State machine for module availability based on prerequisites |
51//! | [`storage`] | LocalStorage WASM bindings for progress persistence |
52//! | [`audio`] | Sound effect playback via JavaScript interop |
53//! | [`learn_state`] | Tab focus and inactivity detection for the Learn page |
54//! | [`struggle`] | Detects when users need hints based on attempt patterns |
55//! | [`ui`] | Dioxus components, pages, router, and theme system |
56//!
