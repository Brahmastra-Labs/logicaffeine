Proof panel for Logic mode.

Displays proof derivation trees and provides tactic buttons for guiding proofs. Shows proof status (idle, proving, success, failed) and optional Socratic hints.

§Props

  • proof_text - Formatted derivation tree HTML
  • status - Current proof state
  • hint - Optional Socratic guidance text
  • on_tactic - Callback when a tactic button is clicked

§Tactics

  • Auto: Automatic proof search
  • Modus Ponens: From P→Q and P, derive Q
  • ∀ Elim: Universal instantiation
  • ∃ Intro: Existential introduction
  • Induction: Structural induction on Nat
  • Rewrite: Term substitution using equality
